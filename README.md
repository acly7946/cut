# Cut

This is my re-implementation of cut
It's just for educational/fun purposes

## Building from source
Working on Arch Linux using:
* gcc 10.2.0
* gnu make 4.3
* scdoc 1.11.0

### Make Dependencies
* GCC or equivalent
* make
* scdoc

### Compiling
```
git clone https://github.com/acly7946/cut.git
cd cut
make
# To install
sudo make install
# To uninstall
sudo make uninstall
```
